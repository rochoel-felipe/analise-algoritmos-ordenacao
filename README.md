# Análise Comparativa de Algoritmos de Ordenação

Este repositório contém o código-fonte e os dados brutos utilizados no artigo científico de comparação entre os algoritmos **Bubble Sort**, **Merge Sort** e **Quick Sort**.

## 📂 Estrutura do Projeto
* **src/**: Contém o código-fonte principal em C (`analise_algoritmos.c`).
* **data/**: Contém os dados textuais brutos das rodadas de testes.
* **docs/**: Contém o artigo final em formato PDF.

## 💻 Ambiente de Execução
* **Processador:** AMD Ryzen 5 5600G
* **Linguagem:** C (Compilador GCC)
* **Sistema Operacional:** Windows 11
